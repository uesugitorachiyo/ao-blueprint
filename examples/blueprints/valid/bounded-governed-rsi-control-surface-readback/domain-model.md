# Domain Model

- `bounded_governed_rsi`: evidence-backed local improvement loop with explicit human/policy boundaries.
- `full_autonomous_self_mutating_rsi`: denied claim level until a separate Covenant-approved publication process exists.
- `chain-binding readback`: observer-only evidence linking Blueprint, Foundry, Forge, Covenant, AO2, and control-plane.
- `GoalRun`: Forge-owned durable state for governed work.
- `readback`: control-plane output that observes evidence and never grants authority.
